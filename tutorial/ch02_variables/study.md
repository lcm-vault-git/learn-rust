# Ch02: 변수와 타입

## 1. 불변 변수 (기본)
```rust
let x = 5;          // 변경 불가
```

## 2. 가변 변수
```rust
let mut y = 5;      // mut 붙이면 변경 가능
y = 10;
```

## 3. 상수
```rust
const MAX: u32 = 100;   // 반드시 타입 명시, 대문자_스네이크
```

> let vs const 차이: const는 컴파일 타임에 결정, 항상 타입 필요, mut 불가

## 4. 섀도잉
```rust
let z = 5;
let z = z + 1;      // 같은 이름으로 재선언 OK (새 변수)
let z = z * 2;       // z = 12
```

> mut과 다름! 섀도잉은 타입도 바꿀 수 있음:
```rust
let spaces = "   ";       // &str
let spaces = spaces.len(); // usize (타입 변경!)
```

## 5. 기본 타입

### 숫자 타입 읽는 법
- 앞글자: `i` = signed(음수 가능), `u` = unsigned(양수만), `f` = float(소수점)
- 뒷숫자: 비트 수 (메모리 크기)
- C++의 `int` = `i32`, `unsigned int` = `u32`, `double` = `f64`

| 타입 | 범위 | 언제 쓰나 |
|------|------|----------|
| `i32` | -21억 ~ 21억 | **기본 정수. 대부분 이거 쓰면 됨** |
| `u32` | 0 ~ 42억 | 음수가 절대 없을 때 (나이, 점수 등) |
| `u8` | 0 ~ 255 | 바이트 데이터 (파일, 네트워크) |
| `i64` | 매우 큰 정수 | 타임스탬프 등 큰 수 필요할 때 |
| `usize` | 0 ~ (시스템에 따라) | 배열 인덱스, 길이 (.len()의 반환 타입) |
| `f64` | 소수점 (64비트) | **기본 실수. 대부분 이거** |
| `f32` | 소수점 (32비트) | 정밀도 낮아도 될 때 |
| `bool` | true / false | 조건 판단 |
| `char` | `'A'` (작은따옴표) | 유니코드 문자 1개 |
| `&str` | `"hello"` (큰따옴표) | 문자열 슬라이스 (읽기 전용) |

> 실무 팁: 타입을 안 쓰면 Rust가 알아서 추론함.
> `let x = 5;`는 자동으로 i32, `let y = 3.14;`는 자동으로 f64

## 6. 타입 추론
```rust
let a = 100;        // i32로 추론
let b = 3.14;       // f64로 추론
let c = true;       // bool로 추론
```

## 7. 튜플
```rust
let tup = (1, 2.0, 'a');
println!("{}", tup.0);   // 인덱스로 접근 (0부터)
```

## 8. 배열
```rust
let arr = [1, 2, 3, 4, 5];
println!("{}", arr[0]);      // 첫 번째 요소
println!("{}", arr.len());   // 길이
```

## 암기 포인트
- 변수는 기본 **불변**, `mut` 붙여야 변경 가능
- `const`는 **대문자 + 타입 필수**
- 섀도잉 = 같은 이름 `let` 재선언 (타입 변경 가능)
- 정수 기본 `i32`, 실수 기본 `f64`
- 문자 `char`는 **작은따옴표**, 문자열 `&str`는 **큰따옴표**
